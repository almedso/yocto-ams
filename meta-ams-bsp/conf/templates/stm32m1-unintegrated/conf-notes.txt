Welcome to almedso BSP-Yocto Linux buildsystem.

Default distro: ams-bsp
Default machine: stsm32mp15-disco (maps to eval board STM32MP157F-DK2)

Run

   bitbake bsp-image-dev

Documentation from STM:
  https://wiki.st.com/stm32mpu/wiki/Category:STM32MPU_Embedded_Software_distribution
User manual of Stm32mp157f-dk2 Evaluation board
  https://www.st.com/resource/en/user_manual/dm00652742.pdf
Data sheet of stm32mp157f
  https://www.st.com/resource/en/datasheet/stm32mp157f.pdf

Alternative OpenSTLinux (layer) images:

  - Official OpenSTLinux images: (distro openstlinux-weston)
      st-image-weston       -   OpenSTLinux weston image with basic Wayland support (if enable in distro)

  - Other OpenSTLinux images:
      - Supported images:
          st-image-core         -   OpenSTLinux core image

Flash the result via

  - Go to image deploy directory of stm32mp15-disco
        cd tmp-glibc/deploy/images/stm32mp15-disco/
  - Create a raw image considering the flash layout and 'dd' the raw image to sdcard
  		./scripts/create_sdcard_from_flashlayout.sh flashlayout_bsp-image-dev/extensible/FlashLayout_sdcard_stm32mp157f-dk2-extensible.tsv
